Component Where How to install Notes
Claude Code Local npm install -g @anthropic-ai/claude-code One install per machine. Generates the JSONL session logs.
Sync agent Local git clone Aimprint && npm install
npm run sync (cron or manual)		 Reads ~/.claude/projects/*.jsonl, classifies sessions, pushes to Worker. Needs a sync token from Settings.
Cloudflare Worker + D1 Cloud npm run worker:deploy
(needs CLOUDFLARE_API_TOKEN)		 Deploy once. Auto-scales. D1 is a managed SQLite — no server to maintain.
Firebase Auth Cloud Configured in Firebase Console once Google SSO — no user management to maintain.
React Dashboard Cloud Browser git push → GitHub Actions builds + deploys automatically Hosted on GitHub Pages. No server. Accessible from any browser.
Data flow — Sync
1. Claude Code writes JSONL after each API request
2. Sync agent reads from cursor offset (never re-reads)
3. Classifier assigns work type (code_writing, planning…)
4. Batch POST to Worker with X-Sync-Token
5. Worker deduplicates by request_id
Data flow — Dashboard
1. User opens app → Firebase Google SSO
2. Firebase issues JWT (ID token)
3. React sends Bearer JWT on every API call
4. Worker verifies JWT → queries D1 filtered by user_id
5. Charts rendered client-side via recharts
D1 Schema
token_usage — one row per API request
session_meta — category, first_message, tool_summary
sync_tokens — one per machine, created in Settings
users — Firebase UID → email mapping

All queries filter by user_id (multi-tenant safe)
