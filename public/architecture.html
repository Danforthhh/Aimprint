What to install where

Component Where How to install / deploy Notes
Claude Code Local npm install -g @anthropic-ai/claude-code One install per machine. Auto-generates JSONL logs at ~/.claude/projects/
Sync agent Local git clone Aimprint → npm install → add sync token in Settings → npm run sync (cron or manual) Reads local JSONL, classifies work types, pushes to Worker. Each machine needs its own sync token.
Cloudflare Worker + D1 Cloud CLOUDFLARE_API_TOKEN=$(cat /c/Code/.cloudflare-token) npm run worker:deploy Deploy once (or on worker/ changes). D1 is a managed SQLite — no server to maintain.
Firebase Auth Cloud Configured once in Firebase Console — Google provider enabled Fully managed. No user database to maintain.
React Dashboard Cloud Browser git push → GitHub Actions builds + deploys automatically Hosted on GitHub Pages. No server. Works from any browser on any device.
